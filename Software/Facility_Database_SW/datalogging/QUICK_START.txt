QUICK START GUIDE - RFID Authentication System
==============================================

SETUP (Do Once):
1. Visit: http://your-domain/datalogging/setup_auth.php
2. Login at: http://your-domain/datalogging/login.php
   Username: admin
   Password: admin123
3. Go to User Management and CHANGE THE PASSWORD!
4. Delete setup_auth.php after setup

USER ROLES:
- admin       = Full access, can manage users
- accounting  = Access to accounting reports
- reception   = Access to reception reports
- MoD         = Normal user access

PROTECT A PAGE:
Add this as the first line after <?php:
    include 'auth_check.php';

PROTECT WITH ROLE:
    include 'auth_check.php';
    requireRole('admin');  // or 'accounting' or 'reception' or 'MoD'
    requireRole(['admin', 'accounting']);  // multiple roles
